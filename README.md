# Towriss-et-al.-2026
SCFA analysis for manuscript

R code for the following analysis:

1. Water consumption linear modeling and posthoc comparisons of slopes over time
2. Body weight linear modeling and posthoc comparisons of slopes over time
3. Sickness scoring of sickness responses 3hrs after LPS injection. Analyzed with ART and corrected posthocs
4. WGCNA network construction, correlation to traits and module enrichments
5. Differential Gene Expression analysis of Diet x LPS effects within each Sex
6. Ranked Gene List comparison analysis
7. GSEA analysis using gene sets from MGEnrichment
8. Microglia Morphology analysis using MicrogliaMorphologyR
9. Microglial Density ANOVA and corrected posthoc tests
